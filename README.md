# launch-demo
:tada: 欢迎使用 launch-demo 项目！ :tada: 

项目使用Claude 4.5 Sonnet和launch进行开发和搭建。

## 部署至launch平台的方式

1. .zip包部署
   - 将项目文件打包成.zip格式。
   - 登录launch平台，选择“上传.zip包”进行部署。
2. GitHub仓库部署
   - 将项目代码推送到GitHub仓库。
   - 在launch平台选择“从GitHub仓库部署”，输入仓库地址进行部署。

不需要npm run build打包生成dist文件，需要的是整个项目的zip包（包括node_modules）。
